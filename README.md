# UMB League Manager 

Proyecto académico desarrollado para la asignatura **Taller de Programación** en la **Universidad Manuela Beltrán**.

##  Descripción del Proyecto

**UMB League Manager** es un sistema desarrollado en **Python** que permite gestionar equipos deportivos semiprofesionales dentro de la universidad y organizar torneos internos.

El sistema facilita la administración de equipos, generación automática de enfrentamientos, registro de resultados y cálculo de la tabla de posiciones.

La aplicación funciona en **modo consola** y está diseñada utilizando **Programación Orientada a Objetos (POO)**.

---

##  Objetivo

Desarrollar una herramienta tecnológica que permita mejorar la **organización, visibilidad y gestión de actividades deportivas universitarias**, facilitando el registro de equipos y la administración de torneos internos.

---

##  Tecnologías utilizadas

* Python 3
* Programación Orientada a Objetos (POO)
* Consola interactiva

##  Funcionalidades

El sistema permite:

* Registrar equipos deportivos
* Crear torneos internos
* Generar enfrentamientos automáticamente
* Registrar resultados de partidos
* Calcular puntos y estadísticas
* Mostrar tabla de posiciones
* Determinar el equipo campeón
